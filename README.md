# Syllabify

Syllabify is a web-based application that helps students transform course syllabi into structured study plans and schedules.

## Tech Stack
- Frontend: React
- Backend: Python (FastAPI or Flask)
- Database: MySQL
- CI/CD: GitHub Actions
